# lostnfoundproject

Systems used:
```

```

RUN IN TERMINAL AND COPY&PASTE THE FOLLOWING:
~~~
mkdir cs111

cd cs111/

git clone https://github.com/Dmokrit/lostnfound.git

cd cs111/lostnfound

python -m venv venv 

source venv/bin/activate.fish

pip install -r requirements.txt

python app.py
~~~
